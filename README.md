# تطبيق دريم الجديد - Dream New App

تطبيق Angular شامل لإدارة المطاعم والتوصيل مع دعم الذكاء الاصطناعي.

## حالة المشروع الحالية ✅

### ✅ تم إصلاحه:
- **التنقل**: إصلاح مشاكل RouterLink و RouterLinkActive
- **المنفذ**: تغيير من 3000 إلى 4200
- **التبعيات**: تحديث وإصلاح الثغرات الأمنية (من 8 إلى 5 ثغرات متبقية)
- **Cypress**: تثبيت وإعداد اختبارات E2E
- **البناء**: التطبيق يعمل بنجاح على localhost:4200

### ⚠️ مشاكل متبقية:
- **Linting**: 30 تحذير متبقي
- **الاختبارات**: 32 اختبار فاشل (مشكلة TestBed.initTestEnvironment)
- **الأمان**: 5 ثغرات أمنية متبقية (xlsx عالية الخطورة، ip-address متوسطة)

## تشغيل محلياً

**المتطلبات:** Node.js

1. تثبيت التبعيات:
   ```bash
   npm install
   ```

2. تشغيل الخادم:
   ```bash
   npm run dev
   ```

3. الوصول للتطبيق على: http://localhost:4200

## هيكل المشروع

```
src/
├── app/
│   ├── components/     # المكونات
│   ├── services/       # الخدمات
│   ├── models/         # النماذج
│   └── utils/          # الأدوات المساعدة
├── public/             # الملفات العامة
```

## الأوامر المتاحة

```bash
npm run dev          # تشغيل الخادم المحلي
npm run build        # بناء التطبيق للإنتاج
```
